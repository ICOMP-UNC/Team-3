Detailed Description

Button control and debounce handling for STM32.

This file provides functions and definitions for initializing buttons on GPIO ports, reading button states, and managing debounce for reliable button presses. It includes configurations for a STOP button and a CONTROL button, each connected to specific pins on GPIOB. The debounce function helps to eliminate noise from mechanical button presses, ensuring reliable detection of button states.
